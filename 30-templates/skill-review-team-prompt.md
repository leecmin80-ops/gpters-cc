# kkirikkiri 스킬 리뷰 프롬프트

## 카드뉴스 스킬 리뷰

```text
cardnews-maker 스킬을 여러 관점에서 검토해줘.

필요한 관점:
- 콘텐츠 기획자
- 카피라이터
- 디자이너
- QA 리뷰어

검토할 것:
- 수강생이 혼자 따라갈 수 있는가
- 카드뉴스 구조가 안정적인가
- 결과물이 너무 길거나 산만해지지 않는가
- 실패했을 때 복구 프롬프트가 충분한가
- 개선안 5개
```

## 주식 뉴스 분석 스킬 리뷰

```text
stock-news-analyst 스킬을 여러 관점에서 검토해줘.

필요한 관점:
- 리서처
- 주식 애널리스트
- 데이터 정리자
- 리스크 검토자

검토할 것:
- 기사 URL 수집 흐름이 이해 가능한가
- 중복/유사 이슈 정리 기준이 명확한가
- 종목 분석 리포트 구조가 실용적인가
- 투자 권유처럼 보이지 않게 안전장치가 있는가
- 개선안 5개
```
