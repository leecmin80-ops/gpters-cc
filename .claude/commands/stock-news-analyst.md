# stock-news-analyst

> 종목명 또는 키워드를 입력하면 뉴스 수집 → 중복 정리 → 월가 스타일 분석 리포트를 자동 생성하는 스킬

---

| 설정 | 기본값 | 변경 방법 |
|------|--------|-----------|
| 수집 건수 | 30건 | 고정 (Phase 2에서 선택 가능) |
| 정렬 | 날짜순 (최신) | 고정 |
| 저장 위치 | `50-my-work/stock-news-analyst/` | 고정 |

## 워크플로우

### Step 0: 이어하기 확인
**타입**: prompt

`50-my-work/stock-news-analyst/` 폴더를 확인한다.
- `01-news-urls.csv`만 있으면 → "뉴스 목록이 있습니다. 중복 정리(Step 3)부터 이어할까요?" 확인
- 3개 파일 모두 있으면 → "기존 리포트가 있습니다. 새로 분석할까요, 기존 리포트를 확인할까요?" 확인
- 파일 없으면 → Step 1로 진행

### Step 1: 키워드 입력
**타입**: prompt

사용자에게 종목명 또는 키워드를 받는다.

입력 검증:
- 키워드가 비어있으면 → "분석할 종목명이나 키워드를 입력해주세요. 예: '삼성전자', '2차전지', '엔비디아'" 안내 후 재입력

### Step 2: 뉴스 수집
**타입**: api_mcp

NaverSearch MCP의 `search_news`를 사용해서 최신 뉴스를 수집한다.

수집 방법:
1. `mcp__claude_ai_PlayMCP__NaverSearch-search_news`로 15건 수집 (start=1)
2. 같은 도구로 추가 15건 수집 (start=16)
3. 총 30건을 아래 형식의 CSV로 정리

CSV 컬럼:
- 제목, 언론사URL, 네이버URL, 핵심내용, 시간

NaverSearch MCP가 연결되지 않은 경우:
- WebSearch로 "site:news.naver.com {키워드}" 검색으로 fallback
- fallback 사용 시 "NaverSearch MCP 미연결로 WebSearch를 사용합니다" 안내

파일 저장 전 기존 파일 확인:
- `01-news-urls.csv`가 이미 있으면 → "기존 뉴스 목록을 덮어쓸까요?" 확인
- 확인 후 `50-my-work/stock-news-analyst/01-news-urls.csv`에 저장

### Step 3: 중복·유사 이슈 정리
**타입**: prompt

수집한 기사들을 분석해 중복·유사 이슈를 묶는다.

묶음 기준 (구체적 규칙):
- 같은 날 같은 사건을 다룬 기사 → 1건으로 묶음
- 제목에 공통 키워드가 2개 이상 → 동일 이슈로 분류
- 같은 증권사 리포트를 인용한 기사 → 원본 리포트 기준 1묶음
- 같은 실적/목표주가 수치를 언급 → 동일 이슈
- 같은 제품/기술 발표를 다룸 → 동일 이슈
- 같은 시장 반응/수급 이슈 → 동일 이슈

출력 형식:
- 이슈 번호, 이슈 제목, 핵심 요약 1줄, 관련 기사 목록
- 목표: 30건 → 5~7개 이슈 묶음

파일 저장:
- `50-my-work/stock-news-analyst/02-deduped-news.md`

### Step 4: 분석 리포트 작성
**타입**: generate

정리된 이슈를 바탕으로 월가 애널리스트 스타일 리포트를 작성한다.

리포트 필수 섹션:

| 섹션 | 내용 |
|------|------|
| Executive Summary | 전체 뉴스 흐름 3줄 요약. 핵심 thesis 제시 |
| 핵심 이슈 분석 | 이슈별: 현황 테이블 + 뉴스 기반 관점 요약 (2~3줄 해석, AI 자동 분류임을 명시) |
| 긍정 요인 | 뉴스에서 확인된 긍정적 시그널 (요인 + 근거) |
| 부정 요인 | 뉴스에서 확인된 리스크 시그널 (요인 + 근거) |
| 주목할 변수 | 아직 결론이 나지 않은 이슈 — 규제 심사 중, 실적 발표 대기 등 (요인 + 확인 시점) |
| Key Metrics to Watch | 모니터링 지표 + 시점 테이블 |
| 후속 검색 키워드 | 다음에 확인할 키워드 5개 |
| Disclaimer | 아래 문구 반드시 포함 |

Disclaimer 문구 (고정):
```
본 리포트는 AI가 공개 뉴스 기사를 자동 수집·분류한 정보 정리 자료이며,
전문 애널리스트의 의견이 아닙니다. 투자 권유가 아니며,
매수·매도 판단은 투자자 본인의 책임입니다.
본 자료의 정확성·완전성을 보장하지 않습니다.
투자 결정 전 공시 자료와 전문가 상담을 권장합니다.
```

톤 & 스타일:
- 영문 섹션 제목 + 한국어 본문 혼합 (월가 리포트 스타일)
- 이슈별 "뉴스 기반 관점 요약"에서 시장 영향을 중립적으로 해석
- 숫자와 데이터를 근거로 제시
- 절대 매수/매도 추천하지 않음

파일 저장:
- `50-my-work/stock-news-analyst/03-stock-news-report.md`

### Step 5: 완료 안내
**타입**: prompt

완료 메시지:
```
종목 분석 완료!

키워드: {입력 키워드}
수집 기사: {N}건 → {M}개 이슈로 정리

저장된 파일:
- 뉴스 목록: 50-my-work/stock-news-analyst/01-news-urls.csv
- 이슈 정리: 50-my-work/stock-news-analyst/02-deduped-news.md
- 분석 리포트: 50-my-work/stock-news-analyst/03-stock-news-report.md

다른 키워드로 다시 분석하려면: /stock-news-analyst
```

## 절대 하지 마

- 매수·매도 추천, 목표가 제시, 수익 보장 표현을 하지 마
- 특정 종목에 고정된 스킬로 만들지 마 — 키워드는 항상 사용자 입력값
- 기사 본문 전체를 크롤링하지 마 — 제목 + description으로 분석
- Disclaimer를 생략하지 마
- 사용자 파일을 확인 없이 덮어쓰지 마
- "투자하세요", "매수 적기", "급등 예상" 같은 표현을 쓰지 마
