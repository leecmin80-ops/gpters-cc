# Clip 5. 카드뉴스 생성 스킬 만들기

## 목표

| 항목 | 내용 |
|---|---|
| 만들 스킬 | `cardnews-maker` |
| 목적 | 주제/자료를 카드뉴스 기획서와 HTML 카드뉴스로 바꾸는 흐름을 스킬화 |
| 핵심 도구 | `show-me-the-prd`, `skillers-suda` |
| 결과물 | 카드뉴스 PRD, 스킬 초안, 샘플 카드뉴스 |

## 진행 순서

| Step | 할 일 | 산출물 |
|---:|---|---|
| 1 | 카드뉴스 구조를 먼저 질문 | 구조 옵션 |
| 2 | 주제 또는 자료를 정한다 | 입력 자료 |
| 3 | 카드뉴스 기획서를 만든다 | `01-cardnews-plan.md` |
| 4 | HTML 카드뉴스를 만든다 | `02-cardnews.html` |
| 5 | 이 흐름을 PRD로 정리한다 | PRD 문서 |
| 6 | `skillers-suda`로 스킬을 만든다 | `cardnews-maker` |
| 7 | 다른 주제로 재실행한다 | 재사용 확인 |

## 실습 프롬프트 1 — 구조 질문

```text
카드뉴스 만들고 싶어.
바로 만들지 말고 카드뉴스가 보통 어떤 형태와 구조로 만들어지는지 알려줘.

특히 아래 기준으로 정리해줘.
- 인스타그램 카드뉴스 기준
- 10장 구성 기준
- Hook / 본문 / 정리 / CTA 역할
- 한 장에 들어갈 텍스트 분량
- HTML로 만들 때 주의할 점
```

## 실습 프롬프트 2 — 기획서 생성

```text
좋아. 그럼 [주제 또는 자료]를 바탕으로
인스타그램 카드뉴스 10장 기획서를 만들어줘.

각 카드마다 아래 항목을 포함해줘.
- 카드 번호
- 카드 역할
- 제목
- 본문
- 시각 요소
- 디자인 메모

결과는 01-cardnews-plan.md로 저장해줘.
```

## 실습 프롬프트 3 — HTML 생성

```text
01-cardnews-plan.md 기획서대로 02-cardnews.html을 만들어줘.

조건:
- 한 파일 안에 10장
- 1080x1080 정사각 카드
- 세로 스크롤
- 한글 가독성 좋은 폰트
- 카드 번호 표시
- 모바일에서도 확인 가능
```

## 실습 프롬프트 4 — PRD 만들기

```text
방금 진행한 카드뉴스 제작 흐름을
사용자가 주제나 자료를 입력하면 반복 실행할 수 있는
cardnews-maker 스킬로 만들기 위한 PRD로 정리해줘.

show-me-the-prd 형식으로 만들어줘.
```

## 실습 프롬프트 5 — 스킬 만들기

```text
이 PRD를 바탕으로 skillers-suda를 사용해서
cardnews-maker 스킬을 만들어줘.

이 스킬은 사용자가 주제나 자료를 입력하면
카드뉴스 구조 제안, 기획서 작성, HTML 카드뉴스 생성,
검토 체크리스트까지 안내해야 해.
```

## 완료 기준

| 항목 | 완료 조건 |
|---|---|
| 구조 질문 | 카드뉴스 구성 원칙을 받았다 |
| 기획서 | `01-cardnews-plan.md`가 있다 |
| HTML | `02-cardnews.html`이 있다 |
| PRD | `cardnews-maker` 요구사항이 정리됐다 |
| 스킬 | `cardnews-maker` 초안이 생성됐다 |
| 재사용 | 다른 주제로 다시 실행할 수 있다 |
