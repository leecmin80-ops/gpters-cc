# GPTers Claude Code 스터디 1주차 가이드

## 목표

| 구분 | 내용 |
|---|---|
| 오늘의 핵심 | 반복 가능한 작업 흐름을 Claude Code 스킬로 만든다. |
| 실습 1 | 카드뉴스 생성 스킬 `cardnews-maker`를 만든다. |
| 실습 2 | 주식 뉴스 분석 흐름을 `stock-news-analyst` 스킬로 만든다. |
| 리뷰 | kkirikkiri로 두 스킬을 점검한다. |
| 과제 | 토요일 오프라인 모임에서 웹앱을 기획, 구현, 배포한다. |

## 클립

| 순서 | 클립 | 핵심 산출물 |
|---:|---|---|
| 0 | 오프닝 | 진행 순서 확인 |
| 1 | 하네스 엔지니어링 | Claude Code를 의도대로 동작시키는 작업 환경 |
| 2 | alias 및 실행 모드 설정 | `cc`, `ccd`, `ccr` |
| 3 | hook 설정 | 알림 hook, 위험 명령 차단 hook |
| 4 | GPTaku Plugin 설치 | `show-me-the-prd`, `skillers-suda`, `kkirikkiri`, `insane-search` |
| 5 | 카드뉴스 생성 스킬 | `cardnews-maker` |
| 6 | 주식 뉴스 분석 스킬 | `stock-news-analyst` |
| 7 | kkirikkiri 리뷰 | 두 스킬 개선안 |
| 8 | 토요일 과제 | 웹앱 제작 과제 안내 |
