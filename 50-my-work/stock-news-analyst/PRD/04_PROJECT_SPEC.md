# stock-news-analyst -- 프로젝트 스펙

---

## 기술 스택

| 영역 | 선택 | 이유 |
|------|------|------|
| 런타임 | Claude Code Skill | 스킬 형태로 반복 실행 가능 |
| 뉴스 수집 | NaverSearch MCP (search_news) | 한국어 뉴스 최적, API 키 불필요 |
| 출력 | Markdown + CSV | 읽기/수정 쉽고 버전 관리 가능 |

---

## 절대 하지 마 (DO NOT)

- [ ] 매수·매도 추천, 목표가 제시, 수익 보장 표현을 하지 마
- [ ] 특정 종목(예: 하이닉스)에 고정된 스킬로 만들지 마 — 키워드는 항상 사용자 입력값
- [ ] 기사 본문 전체를 크롤링하지 마 — 제목 + description으로 분석
- [ ] Disclaimer를 생략하지 마 — 리포트 마지막에 반드시 포함
- [ ] 사용자 파일을 확인 없이 덮어쓰지 마
- [ ] "투자하세요", "매수 적기", "급등 예상" 같은 표현을 쓰지 마

---

## 항상 해 (ALWAYS DO)

- [ ] 키워드를 입력받는 것으로 시작
- [ ] 뉴스 수집 시 날짜순(date) 정렬
- [ ] 중복 정리 시 이슈별 묶음 기준 명시
- [ ] 리포트에 Executive Summary, Bull/Bear Case, Key Metrics 포함
- [ ] 리포트 마지막에 "투자 권유가 아님" Disclaimer 명시
- [ ] 후속 검색 키워드 5개 제안
- [ ] 결과물을 `50-my-work/stock-news-analyst/`에 저장
