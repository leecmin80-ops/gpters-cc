# cardnews-maker -- 데이터 모델

> 이 문서는 스킬에서 다루는 핵심 데이터의 구조를 정의합니다.

---

## 전체 구조

```
[입력(Input)] --생성--> [기획서(Plan)] --생성--> [HTML 카드뉴스]
                              |
                              └--적용--> [디자인 테마(Theme)]
```

---

## 엔티티 상세

### 입력 (Input)
사용자가 스킬 호출 시 전달하는 정보

| 필드 | 설명 | 예시 | 필수 |
|------|------|------|------|
| topic | 카드뉴스 주제 | "청소 꿀템 소개" | O |
| source | 참고 자료 | URL, 텍스트, 파일 경로 | X |
| theme | 디자인 테마 이름 | "mint", "warm", "dark" | X (기본: mint) |

### 기획서 (Plan)
카드 1장의 기획 정보. 10장이 하나의 기획서 파일을 구성.

| 필드 | 설명 | 예시 | 필수 |
|------|------|------|------|
| card_no | 카드 번호 (1~10) | 1 | O |
| role | 카드 역할 | "Hook", "본문", "정리", "CTA" | O |
| title | 카드 제목 | "청소 고수들만 아는 꿀템 7가지" | O |
| body | 본문 텍스트 | 2~3줄 분량 | O |
| visual | 시각 요소 설명 | "반짝이는 깨끗한 방 배경" | O |
| design_note | 디자인 메모 | "민트/화이트 톤, 32pt 볼드" | X |

### 디자인 테마 (Theme)
HTML 카드뉴스에 적용할 시각 스타일 세트

| 필드 | 설명 | 예시 | 필수 |
|------|------|------|------|
| name | 테마 이름 | "mint-fresh" | O |
| colors | 주요 색상 3개 | #00b894, #ffffff, #333333 | O |
| font_style | 폰트 스타일 | "깔끔한 고딕" | O |
| bg_gradient | 배경 그라디언트 방향 | "135deg" | O |

### 검토 결과 (Review)
검토 체크리스트 실행 결과

| 필드 | 설명 | 예시 | 필수 |
|------|------|------|------|
| item | 검토 항목 | "Hook 카드에 호기심 유발 요소 있음" | O |
| status | 통과 여부 | pass / warn / fail | O |
| note | 보완 메모 | "CTA 문구 좀 더 구체적으로" | X |

---

## 내장 테마 3종

| 테마 | 주요 색상 | 느낌 |
|------|----------|------|
| mint | #00b894, #e0f7f1, #1a1a1a | 깨끗하고 시원한 |
| warm | #e17055, #fff4e6, #2d3436 | 따뜻하고 친근한 |
| dark | #2d3436, #636e72, #ffffff | 세련되고 모던한 |
