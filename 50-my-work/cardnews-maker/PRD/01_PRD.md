# cardnews-maker -- PRD (Product Requirements Document)

---

## 1. 제품 개요

### 한 줄 요약
주제를 입력하면 인스타그램 카드뉴스 기획서와 HTML 카드뉴스를 자동 생성하는 Claude Code 스킬

### 해결하는 문제
카드뉴스를 만들 때마다 구조 설계, 텍스트 작성, HTML 코딩을 매번 처음부터 반복해야 하는 비효율

### 핵심 가치
주제 한 줄만 입력하면 기획서 → HTML까지 일관된 구조로 자동 생성. 디자인 테마 선택으로 매번 다른 느낌 가능.

---

## 2. 사용자

### 주요 사용자
- **누구**: 본인 (개인 사용)
- **상황**: 인스타그램에 올릴 카드뉴스 콘텐츠가 필요할 때
- **목표**: 주제만 정하면 바로 공유 가능한 카드뉴스를 빠르게 만들고 싶음

### 사용자 시나리오
1. Claude Code에서 cardnews-maker 스킬을 호출하고
2. 주제(예: "청소 꿀템 소개")와 테마(예: mint)를 입력하면
3. 10장 기획서(md)와 HTML 카드뉴스가 자동 생성되어 저장됨

---

## 3. 핵심 기능

| 기능 | 설명 | 우선순위 | 복잡도 |
|------|------|----------|--------|
| 기획서 생성 | 주제 → 10장 기획서(카드별 번호, 역할, 제목, 본문, 시각 요소, 디자인 메모) | P1 (MVP) | 간단 |
| HTML 카드뉴스 생성 | 기획서 → 1080x1080 정사각 HTML 카드뉴스 (세로 스크롤, 한글 폰트, 모바일 대응) | P1 (MVP) | 보통 |
| 검토 체크리스트 | 텍스트 분량, 역할 배분(Hook/본문/정리/CTA), CTA 포함 여부 점검 | P1 (MVP) | 간단 |
| 디자인 테마 선택 | mint, warm, dark 3종 내장 테마 중 선택 | P1 (MVP) | 보통 |

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름
```
주제 입력 -> (테마 선택) -> 기획서 생성 -> HTML 생성 -> 검토 체크리스트 -> 저장 완료
```

### 상세 흐름
1. **주제 입력**: 사용자가 카드뉴스 주제를 텍스트로 입력 (예: "청소 꿀템 소개")
2. **테마 선택**: mint / warm / dark 중 선택 (기본값: mint)
3. **기획서 생성**: 10장 기획서를 `01-cardnews-plan.md`로 저장
4. **HTML 생성**: 기획서 기반 HTML 카드뉴스를 `02-cardnews.html`로 저장
5. **검토 체크리스트**: 생성 결과의 품질을 항목별로 점검
6. **브라우저 확인**: HTML 파일을 브라우저에서 열어 최종 확인

---

## 5. 성공 기준

- [ ] 주제 입력만으로 기획서 + HTML이 자동 생성된다
- [ ] 생성된 카드뉴스가 인스타그램 1080x1080 규격에 맞는다
- [ ] 다른 주제로 바꿔도 같은 품질로 다시 생성된다
- [ ] 3종 테마가 각각 구별되는 디자인으로 적용된다
- [ ] 검토 체크리스트로 누락 항목을 발견할 수 있다
