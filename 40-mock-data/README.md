# 40-mock-data

실습 시연이나 스킬 검증에 사용할 샘플 입력을 두는 폴더다.

## 사용 원칙

- 실제 개인정보, 계정 정보, 유료 데이터는 넣지 않는다.
- 카드뉴스 실습에는 공개 자료 또는 짧은 더미 텍스트를 사용한다.
- 주식 뉴스 분석 실습에는 사용자가 입력한 키워드와 공개 뉴스 URL만 사용한다.
- 검증용 파일은 실제 결과물과 구분되도록 파일명에 `sample-` 또는 `mock-`을 붙인다.

## 예정 샘플

| 파일 예시 | 용도 |
|---|---|
| `sample-cardnews-topic.md` | 카드뉴스 기획 실습용 주제 |
| `sample-news-urls.csv` | 뉴스 중복 정리 실습용 더미 기사 목록 |
